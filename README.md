# lamp-server-dockerfile

Installation

   docker pull iwphox/lamp-server
    Run the docker-compose up -d


cd docker-compose-lamp/
cp sample.env .env
// modify sample.env as needed
docker-compose up -d
// visit localhost

Your LAMP stack is now ready!! You can access it via http://localhost.
